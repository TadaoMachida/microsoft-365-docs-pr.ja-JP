# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>個人用電子メール アドレスを使用してOffice 365にサインアップする

## <a name="when-you-skip-adding-a-custom-domain"></a>カスタム ドメインの追加をスキップする場合


個人用電子メール アドレスを使用してOffice 365にサインアップし、[カスタム ドメインの追加を](../get-help-with-domains/what-is-a-domain.md)スキップすると、特定の機能へのアクセスが制限されます。

カスタム ドメインを今すぐ追加することを選択した場合は、次のすべての Premium 機能にアクセスできます。

|**機能**|**個人用メールを使用したOffice 365**|**カスタム ドメインを使用したOffice 365**|
|:-----|:-----|:-----|
|**OneDrive <sup>1、2</sup>**| [個人用OneDrive](https://onedrive.live.com/about/plans/)| [OneDrive for Business](https://onedrive.live.com/about/en-us/business/) |
|**Office アプリケーション: Word、Excel、PowerPoint、OneNote、Outlook、Access (PC のみ)**| はい | はい
|**ビジネス アプリケーション <sup>3</sup> : Microsoft Bookings と MileIQ**| いいえ | はい
|**Microsoft 365 管理センターへのアクセス**| 制限付きアクセス (課金、サポート、ドメインのセットアップ) | はい
|**ユーザーの追加**| いいえ | はい
|**Office 365 サービス: Exchange Online、Skype for Business、Microsoft Teams、SharePoint Online**| いいえ | はい
|**セキュリティツールとコンプライアンス ツール**| いいえ | はい
