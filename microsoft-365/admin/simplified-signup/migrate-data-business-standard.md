# <a name="migrate-data-to-my-microsoft-365-business-standard-subscription"></a>データを Microsoft 365 Business Standard サブスクリプションに移行する

この記事の手順に従って、OneDrive、Outlook、TeamsデータをサブスクリプションにMicrosoft 365 Business Standardします。

> [!IMPORTANT]
> 引き続き、個人アカウントにデータを保持できます。 新しいビジネス メール アカウントを作成してデータを移行しても、個人アカウントのデータは期限切れしません。 すべてのデータを新しいビジネス アカウントに移動するか、一部のデータを移動できます。 たとえば、仕事用ドキュメントをビジネス アカウントに移動できますが、個人用の家族の写真は個人用アカウントに保持できます。

### <a name="import-email"></a>メールをインポートする

別のメール アカウントで Outlook を使用していた場合は、以前のメール、予定表、連絡先を新しい Microsoft アカウントにインポートできます。
  
1. **古いメールをエクスポートする**

    Outlook で、[**ファイル**] \> [**開く&amp;エクスポート**] \> [**インポート/エクスポート**] の順に選択します。

    [**ファイルにエクスポート**] を選択し、手順に従って Outlook データ ファイル (.pst) およびすべてのサブフォルダーをエクスポートします。

2. **古いメールをインポートする**

    Outlook で、[**ファイル**] \> [**開く&amp;エクスポート**] \> [**インポート/エクスポート**] を再度選択します。

    今度は、[**他のプログラムまたはファイルからのインポート**] を選択し、古いメールをエクスポートしたときに作成したバックアップ ファイルを手順に従ってインポートします。

## <a name="move-data-from-your-personal-microsoft-teams-account-to-new-teams-for-work-account"></a>個人用アカウントから仕事用Microsoft Teams新しいTeamsにデータを移動する

1. Microsoft Teams を開き、プロファイル アイコンを選択して、**[職場または学校アカウントを追加する]** を選択します。

2. 手順に従って、新しいアカウントを作業Teams追加します。 詳細については[、「サインイン」を参照し、Teams](https://support.microsoft.com/office/sign-in-and-get-started-with-teams-6723dc43-dbc0-46e6-af49-8a2d1c5cb937)を開始してください。

### <a name="access-teams-chats"></a>チャットTeamsアクセスする

新しい仕事用Teams使用を開始しても、データは移行されません。 古いチャットを確認する最善の方法は、古いアカウントとTeams仕事用アカウントを並べて開く方法です。 この操作を行うには、Teamsの右上にある [ME] アイコンを選択し、開くアカウントを選択します。 新しい職場のアカウントTeams、同僚と一緒に使用を開始できます。 チャットを行う他のユーザーに、新しい仕事用アカウントを使用して連絡を開始Teams確認してください。

### <a name="microsoft-teams-meetings"></a>Microsoft Teams会議

作業用に新しいMicrosoft Teamsアカウントを設定したら、会議の予定表で会議Teamsできます。 古い会議アカウントの元の会議をTeamsしてください。 これにより、スケジュール時の予定表の可用性、会議を記録する機能など、より豊富な機能にアクセスできます。 会議を削除できるのは、自分の予定表Teamsのみなので、会議を持つユーザーに、会議を再作成する予定を知らせる必要があります。 会議に新しい Teams アカウントを使用する移行時に、会議に参加する必要があるユーザーが存在しない場合は、会議に参加していないユーザーに連絡して、古い会議リンクに参加していないか確認してください。

### <a name="migrating-contacts"></a>連絡先の移行

個人用アカウントから連絡先を移行Teams、連絡先の電子メール アドレスを見つけて、ユーザーを仕事用の新しい Teams アカウントに追加します。
