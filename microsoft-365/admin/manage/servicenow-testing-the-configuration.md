# <a name="testing-the-servicenow-configuration"></a>ServiceNow 構成のテスト

アプリケーションで外部システムとの正常な通信が必要な場合は、接続をテストして構成を正常に行う方法の概要を説明します。

サポート統合の構成をテストするには、次Microsoft 365実行します。

1. 管理者として ServiceNow ポータルにログインします。

2. インシデントを開きます。

3. [サポート] タブ **Microsoft 365** をクリックし、[Microsoft 365 インサイト] を選択して **、推奨** されるソリューションが正常に返されたかどうかを確認します。
