# <a name="search-in-the-microsoft-365-admin-center"></a>Microsoft 365 管理センターで検索する

Microsoft 365組織の管理者は、検索を使用してユーザーを検索し、アクションを実行し、さまざまな設定に移動し、ドキュメントを読むことができます。 新しい検索機能により、検索速度が向上し、管理センターのすべてのページから検索できるようになりました。 検索ボックスが管理センターの上部にあるバナー領域に移動しました。 **Alt + S ショートカット** を使用すると、任意のページから検索を使用できます。

:::image type="content" source="../../media/search-bar.png" alt-text="スクリーンショット: Microsoft 365 管理センターの検索バー":::

検索結果は、さまざまなカテゴリに分類されます。 ほとんどのカテゴリは、管理センター内のアイテムです。 たとえば、ユーザー、グループ、共有メールボックス、ドメインなどです。 他のカテゴリでは、移動できる場所、実行できるアクション、変更できるアプリ レベルの設定が表示されます。 また、ドキュメントに関連するカテゴリもあります。

次のセクションでは、検索可能な管理センターのさまざまな領域とカテゴリについて説明します。

## <a name="before-you-begin"></a>開始する前に

管理センターで検索するには、管理者である必要があります。 検索結果の範囲は、ログインしているユーザーが持つ管理者権限です。 たとえば、管理者SharePoint管理センターに領域やカテゴリが表示されない場合、検索には表示されません。

## <a name="users"></a>ユーザー

ユーザーは、表示名、姓、名、ユーザー名、プライマリ電子メール アドレス、または電子メール エイリアスで確認できます。 ユーザーの名前の編集を選択して、ユーザーの詳細を編集します。

:::image type="content" source="../../media/User-search-stock-picture-domain-blanked.png" alt-text="スクリーンショット: ユーザー Felix Henderson を検索する":::

名前の横にある 3 つのドット (その他のアクション) メニューを選択した場合は、パスワードをリセットできます。

:::image type="content" source="../../media/User-search-stock-picture-reset-password-domain-blanked.png" alt-text="スクリーンショット: 検索結果からパスワードをリセットするオプションを使用したユーザー検索":::

**ユーザー検索結果を改善するためのヒント**

- ユーザー検索が前述のプロパティと正確に一致するように、ユーザーの名前を正しくスペルしてください。 たとえば、上記の例では、Jus または Malz は機能しますが、Justin の代わりに Jostin のようにスペルミスはこのユーザーを見つけることができません。
- 完全一致を取得するには、プライマリ電子メール アドレスまたはユーザー名で検索します。

## <a name="groups"></a>グループ

グループ名またはグループの電子メール アドレスでグループを検索できます。 グループを選択し、任意のページからグループを編集できます。

:::image type="content" source="../../media/search-for-groups-mac.png" alt-text="スクリーンショット: 管理センターのグループの検索結果":::

**グループ検索結果を改善するためのヒント**

グループ名のスペルが正しいことを確認します。

## <a name="actions"></a>アクション

M365 管理センターで頻繁に使用されるアクションが含まれているアクション カテゴリを検索できます。 アクションは、システムの動詞と考えてください。 たとえば、任意のページから "パスワードのリセット" を検索し、ユーザーの 1 つ以上のパスワードをリセットすることもできます。 [ユーザーの削除] を検索し、[ユーザーの削除] ページからユーザーを削除できます。

:::image type="content" source="../../media/search-for-actions-mac.png" alt-text="スクリーンショット: 管理センターで実行できるアクションを示す検索結果":::

## <a name="navigation"></a>ナビゲーション

結果は、管理センターの特定のページにすばやく移動する方法を提供します。 たとえば、RBAC を検索すると、Azure AD ロールの [ロール] ページに移動します。

:::image type="content" source="../../media/search-for-navigation-mac.png" alt-text="スクリーンショット: 管理センターのナビゲーション パスを示す検索結果":::

## <a name="settings"></a>Settings

組織に関連するサポートされているアプリ レベルの設定、サブスクライブするサービス、セキュリティとプライバシーの設定を検索します。

:::image type="content" source="../../media/search-for-settings-mac.png" alt-text="スクリーンショット: 管理センターの設定を示す検索結果":::

## <a name="domain"></a>ドメイン

ドメインへのクイック リンクを確認すると、そのドメインの概要ページに移動します。

:::image type="content" source="../../media/search-for-domains-mac.png" alt-text="管理センターで所有されているドメインを示す検索結果。":::

## <a name="documentation"></a>ドキュメント

ドキュメント検索では、検索語句に基づいて関連するヘルプ ドキュメントが提供されます。 トピックをクリックして詳細を確認します。

:::image type="content" source="../../media/search-for-docs-mac.png" alt-text="スクリーンショット: 管理センターでドキュメントの結果を表示する検索":::

## <a name="send-us-feedback"></a>フィードバックを送信する

このセクションを使用して、検索エクスペリエンスに関するフィードバックを送信します。 すべてのフィードバックに返信することはできませんが、すべてのフィードバックを読み、お客様のフィードバックを使用して検索エクスペリエンスを向上させます。 フィードバックにできる限り詳細を入力してください。

:::image type="content" source="../../media/search-feedback-mac.png" alt-text="スクリーンショット: 管理センターで検索に関するフィードバックを送信する":::
