# <a name="microsoft-365-lighthouse-tenants-page-overview"></a>Microsoft 365 Lighthouseテナント ページの概要

> [!NOTE]
> この記事で説明する機能はプレビューで、変更される可能性があります。要件を満たすパートナーだけが [利用できます](m365-lighthouse-requirements.md)。 組織にアカウントが設定されていない場合Microsoft 365 Lighthouse[を参照してください](m365-lighthouse-sign-up.md)Microsoft 365 Lighthouse。

Microsoft 365 Lighthouse左側のナビゲーション ウィンドウで [テナント]を選択してテナント アカウントを管理し、[テナント] ページを開きます。 [テナント] ページには、すべてのテナントの一覧が表示されます。 テナントを選択すると、連絡先の詳細や展開状態などの詳細情報を表示できます。

[テナント] ページには、次のオプションも含まれています。

- **エクスポート:** テナント データをコンマ区切り (Excel) ファイルにエクスポート.csvします。
- **タグの管理:** タグを追加、編集、または削除する場合に選択します。
- **タグの割り当て:** テナントにタグを割り当てる場合に選択します。
- **検索:** キーワードを入力して、リスト内の特定のテナントをすばやく検索します。


:::image type="content" source="../media/m365-lighthouse-tenants-page-overview/tenant-page-overview.png" alt-text="[テナント] ページのスクリーンショット。":::

## <a name="tenant-list"></a>テナントリスト

テナントリストには、契約しているさまざまなテナント (テナントのライトハウスオンボーディング状態など) に関する分析情報が表示されます。 テナントリストでは、テナントにタグを付け、ライトハウス全体でさまざまなフィルターを提供し、ドリルダウンして、特定のテナントとその展開計画の状態について詳しくは説明します。

テナントがライトハウスのオンボーディング [要件](m365-lighthouse-requirements.md)を満たした後、その状態はテナントリスト **に [アクティブ** ] と表示されます。

テナントリストを使用すると、次のことができます。

- アクティブ、非アクティブ、および不適格でテナントを自動的に並べ替えます。
- テナントリストのエクスポート
- タグの割り当てと管理
- 名前でテナントを検索する
- 状態、委任された管理特権 (DAP)、およびタグによってテナントをフィルター処理します。

テナントを無効にするか、タグを表示および管理するには、テナント名の横にある 3 つのドットを選択します。 テナント名を選択するか、テナントに割り当てられたタグのいずれかを選択して、個々のテナントを表示できます。

## <a name="tenant-status"></a>テナントの状態

次の表に、さまざまな状態とその意味を示します。

| 状態                                | 説明                                         |
|---------------------------------------|-----------------------------------------------------|
| Active                                | オンボーディングとデータ フローが開始されました。               |
| 非アクティブ                              | テナントがアクティブではなくなりました。                         |
| プロセス中                            | テナントが検出されましたが、完全にはオンボードされません。         |
| 不適格な委任アクセスが必要 | 委任された管理者特権 (DAP) のセットアップが必要です。 |
| 不適格、不足している必須ライセンス  | テナントに必要なライセンスはありません。              |
| 不適格、ユーザー数超過       | テナントには、許可されているユーザーよりも多くのユーザーがあります。                 |
| 不適格、契約の種類             | テナントには契約が設定されていない。                    |

テナントを非アクティブ化すると、非アクティブ化プロセスが完了するまでテナントに対してアクションを実行できません。 非アクティブ化が完了するために最大 48 時間かかる場合があります。 テナントを再アクティブ化する場合、データが再び表示されるには最大 48 時間かかる場合があります。

## <a name="tenant-tags"></a>テナント タグ

テナントを整理し、既存のビューを簡単にフィルター処理するために、タグを作成してテナントに割り当てることができます。 詳細については、「テナントリストの [管理」を参照してください](m365-lighthouse-manage-tenant-list.md)。

> [!NOTE]
> すべてのテナントで最大 30 のタグを作成できます。


## <a name="tenant-details-page"></a>テナントの詳細ページ

テナントの詳細な情報を表示するには、テナントリストからテナントを選択します。 テナントの詳細ページには、連絡先情報と展開計画の状態が表示されます。


:::image type="content" source="../media/m365-lighthouse-tenants-page-overview/tenant-details-page.png" alt-text="[テナントの詳細] ページのスクリーンショット。":::

### <a name="overview-tab"></a>[概要] タブ

[概要] タブでは、テナントの概要、連絡先情報、およびサービスの使用状況Microsoft 365表示できます。

#### <a name="tenant-overview-card"></a>テナントの概要カード

テナントの概要カードは、テナントアカウントからテナントに関するMicrosoft 365します。

| テナント情報    | 説明|
|-----------------------|------------------|
| Headquarters    | テナントの場所。|
| 業界    |組織の業界。|
| Web サイト    |組織の Web サイト。 データが指定されている場合は、このフィールドを編集できます。|
| 顧客ドメイン    |組織のドメイン。|
| ユーザー総数    |テナントに割り当てられたユーザーの数。 この番号を選択して、そのテナントの [ユーザー] ページを開きます。|
| デバイスの総数|テナントに登録されているデバイスの数。 この番号を選択して、そのテナントの [デバイス] ページを開きます。|

#### <a name="contacts-card"></a>連絡先カード

連絡先カードを使用すると、管理するテナント内の主要な連絡先の情報を次のように入力できます。

- 名前
- タイトル
- Phone
- 電子メール
- メモ

[メモ] セクションは、テナントの主要な情報 (エンゲージメントの設定、場所、タイム ゾーン、組織内の役割に関する詳細など) を記録するために使用できるテキスト フィールドです。

詳細を編集したり、既存の連絡先を削除したりするには、一覧から連絡先名を選択します。 [連絡先 **の編集] ウィンドウ** で、連絡先を編集または削除します。 別の連絡先を追加するには、[+連絡先の **追加] を選択します**。

#### <a name="microsoft-365-usage-card"></a>Microsoft 365カード

ライトハウスは、Microsoft 365サービスの使用状況に関する分析情報を提供します。テナント内でライセンスを取得し、各サービスをアクティブに使用しているユーザーの数を含めます。 Active は、過去 28 日間に少なくとも 1 回サービスにサインインしたユーザーまたはデバイスの数を示します。 変更は、先月以降のアクティブ ユーザーとデバイスの変化を示します。

[Microsoft 365] 使用法カードには、次の 2 つのセクションがあります。

- Microsoft 365 Lighthouse有効なサービス – ライトハウス ポータル内で管理できるサービス。
- 追加Microsoft 365サービス – Microsoft 365 スイートに含まれているが、現時点では Microsoft 365 Lighthouse ポータル内で管理できないサービス。


### <a name="deployment-plans-tab"></a>[展開計画] タブ

[展開計画] タブには、テナントの展開計画の状態が表示されます。 一覧の展開手順は、テナントに適用されるベースラインに基づいて行います。 展開手順の詳細を表示するには、一覧から展開手順を選択します。

[展開計画] タブには、次のオプションも含まれています。

- **エクスポート:** 展開手順のデータをコンマ区切Excel (.csv) ファイルにエクスポートする場合に選択します。
- **更新:** 最新の展開手順データを取得する場合に選択します。
- **検索:** キーワードを入力して、リスト内の特定の展開手順をすばやく見つける。
