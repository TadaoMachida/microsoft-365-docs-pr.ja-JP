# <a name="important-information-for-office-365-e4-customers"></a>Office 365 E4 のお客様に関する重要な情報

Office 365 E4 プランは 2015 年に廃止されました。 Microsoft 365グローバル管理者は、メールの更新を受け取り、メッセージ <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">センター</a> (Microsoft 365 管理センターの一部) に、アクションを実行する必要があるタイミングに関する情報を含む投稿を表示します。

## <a name="ready-to-upgrade"></a>アップグレードする準備はできましたか?

サブスクリプションを更新する場合は、これらのプランのいずれかにアップグレードすることを検討してください。

Microsoft から直接サブスクリプションを購入し、今すぐアップグレードする場合は、「[Office 365 E4 サブスクリプションからのアップグレード](upgrade-Office-365-E4.md)」を参照してください。 E4 から新しいプランにアップグレードしても、データは失われません。
