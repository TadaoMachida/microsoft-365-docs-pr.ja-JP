# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>Microsoft 365 for business プロダクト キーに関する問題

**Microsoft 365** のプロダクト キーをビジネス用に使用しようとするときにエラー メッセージが表示される場合は、次の一般的なソリューションを試してください。
  
 **すぐにヘルプが必要な場合** [Microsoft サポートに問い合わせください](../admin/get-help-support.md)。
  
 **Microsoft 365 ファミリまたは Microsoft 365 個人用** プロダクト キーのヘルプについては、「Microsoft 365 ファミリまたは Microsoft 365 個人用プロダクト キーを使用する」 [を参照](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759)Office。
