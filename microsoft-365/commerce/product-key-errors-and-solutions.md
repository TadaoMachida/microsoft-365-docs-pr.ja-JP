# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>ビジネスプロダクト キーのMicrosoft 365に関する問題

**ビジネス向けのMicrosoft 365** のプロダクト キーを引き換えようとしたときにエラー メッセージが表示される場合は、次の一般的な解決策を試してください。
  
 **すぐにヘルプが必要な場合** [Microsoft サポートを呼び出します](../admin/get-help-support.md)。
  
 **Microsoft 365 FamilyまたはMicrosoft 365 Personalプロダクト キーのヘルプについては**、「[Officeでのプロダクト キーの使用](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759)」を参照してください。
