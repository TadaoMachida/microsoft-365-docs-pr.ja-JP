# <a name="what-the-dlp-policy-templates-include"></a>DLP ポリシー テンプレートに含まれるもの

[!include[Purview banner](../includes/purview-rebrand-banner.md)]

Microsoft Purview コンプライアンス ポータルのMicrosoft Purview データ損失防止 (DLP) には、米国医療保険法 (HIPAA)、米国グラムリーチブリリー法 (GLBA)、米国愛国者法の対象となる機密情報の保護を支援するなど、一般的なコンプライアンス要件に対応するすぐに使用できるポリシー テンプレートが含まれています。この記事では、すべてのポリシー テンプレート、検索する機密情報の種類、既定の条件とアクションを一覧表示します。 この記事には、各ポリシー テンプレートの構成方法の詳細は含まれていません。代わりに、シナリオの出発点として最適なテンプレートを決定するのに役立つ十分な情報が記事に表示されます。 これらのポリシー テンプレートは、特定の要件を満たすようにカスタマイズできることに注意してください。
