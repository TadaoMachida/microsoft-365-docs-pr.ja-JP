# <a name="microsoft-managed-desktop-service-description"></a>Microsoft マネージド デスクトップ サービスの説明

Microsoft Managed Desktop は、ユーザーにセキュリティで保護されたモダン エクスペリエンスを提供し、Windows 10 Enterprise エディション、Microsoft 365 Apps for enterprise、Microsoft セキュリティ サービスの最新バージョン (以下を含む) を常に最新のバージョンでデバイスに保持します。

- 新しいデバイスの簡単な登録。
- デバイスの構成。
- Microsoft のベスト プラクティスに従って、Windows Hello、BitLocker、SecureBoot、仮想化ベースのセキュリティなど、ユーザーとデバイスをセキュリティ保護するための機能。
- デバイス セキュリティの監視と修復サービス。
- App Assure によるアプリ [の互換性](/fasttrack/products-and-capabilities#app-assure)。
- アプリとアプリのWindows 10のMicrosoft 365 Apps for enterprise管理。
- デバイスとアプリの使用状況に関する分析データ。
- ユーザーの IT サポート。
- IT プロの運用サポート。
