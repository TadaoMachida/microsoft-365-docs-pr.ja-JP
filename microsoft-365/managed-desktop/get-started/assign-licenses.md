# <a name="assign-licenses"></a>ライセンスを割り当てる

Microsoft Managed Desktop に登録する準備として、必要なライセンスを取得している必要があります。 ライセンスの詳細については、「 [必要なライセンスの](../get-ready/prerequisites.md#more-about-licenses) ライセンスの詳細」を参照してください。

適切なライセンスがある場合は、ユーザーに割り当てる必要があります。 ライセンスを割り当てるには、グループベースのライセンス[](/azure/active-directory/fundamentals/active-directory-licensing-whatis-azure-portal)機能を利用することをお勧Azure Active Directory。

ライセンスの割り当てに問題がある場合は、管理者サポートにお問い [合わせください](../working-with-managed-desktop/admin-support.md)。

1. [登録後に設定を調整する](conditional-access.md)。
1. [Intune ポータル サイト](company-portal.md)を展開して割り当てます。
1. ライセンスの割り当て (この記事)。
1. [アプリを展開する](deploy-apps.md)。
1. [デバイスをセットアップする](set-up-devices.md)。
1. [Autopilot と登録ステータス ページの初回実行時エクスペリエンス](esp-first-run.md)のセットアップ。
1. [ユーザー サポート機能を有効にする](enable-support.md)。
